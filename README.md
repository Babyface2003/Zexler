# Zexler
Есть 3 таблицы. Таблица с основной информацией о компаниях, с отзывами сотрудников о компаниях, с информацией по сделанным email-рассылкам по компаниям. Ниже данные представлены только частично для ознакомления со структурой БД.
Company
id

![image](https://github.com/Babyface2003/Zexler/assets/100961971/5809ddb8-11a4-4d8e-ad07-68bcfe042431)



id - уникальный идентификатор компании
name - наименование компании
email - электронная почта компании
rating - рейтинг компании на основе отзывов, где 0 - отсутствие отзывов и рейтинга
prosmotr - количество просмотров страницы с компанией
status_sotrudn - текущий статус сотрудничества с компанией
Feedback

![image](https://github.com/Babyface2003/Zexler/assets/100961971/2416d5a7-a631-4aed-8413-75f653ea4ba1)


id - уникальный идентификатор отзыва
comp_id - id компании из таблицы Company
name - имя оставившего отзыв
date - дата написания отзыва
obsh_vpechatleniya - общие впечатления сотрудника о компании, могут быть двух видов: “рекомендую”, “не рекомендую”
Email_send

![image](https://github.com/Babyface2003/Zexler/assets/100961971/fcabc319-b2e7-4bab-8466-e65305d8ce2c)



id - уникальный идентификатор записи в таблице
campaighn_id - уникальный номер email-рассылки
email - почта компании
status - статус доставки письма (ok_delivered - письмо доставлено, ok_unsubscribed - компания отписалась от рассылок)
date - дата отправки email-рассылки
com_id - id компании из таблицы Company

Задача 1.
Посчитать кол-во компаний, у которых не указана почта в таблице Company. 
Задача 2.
Отфильтровать часто просматриваемые (свыше 50000 просмотров) компании с низким рейтингом по отзывам (ниже 4-х), с которыми мы еще не сотрудничаем.
Задача 3.
Составить Топ 10 компаний, у которых большего всего отзывов. Вывести наименование компании и количество отзывов по ней.
Задача 4.
Отфильтровать компании с высоким рейтингом (от 4-х и выше), у которых самый свежий отзыв сотрудника негативный. Нужны только не партнеры.
Задача 5.
Отфильтровать компании - не партнеры с рейтингом в 2 балла, по которым была сделана только одна рассылка. Исключить компании, которые отписались от рассылки.
